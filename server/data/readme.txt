аз вземам първите 3 категории, Митко вторите 3, Любо последните 3
всеки създава за себе си файл ime.json
и там добавя продуктите като в примера products.json
по 30 продукта на категория

{
    "id": 1,                                   // последователно 1,2,3... за всеки продукт, аз започвам от 1 до 90, Митко от 91 до 180, Любо от 181 до 270
    "name": "name",                            // името на продукти, на кирилица
    "image": "/public/uploads/url",            // всеки създава папка imeImages и там качва основното изображение на артикула и тука пише следния формат: "/public/uploads/ime_na_izobrajenieto.razshirenie"
    "description": "long description here",    // дълго описание на кирилица
    "short_text": "short text here",           // кратък текст, на кирилица
    "price": "12.00",                          // цена, с десетична точка без валута
    "olg_price": "15.00",                      // стара цена, преди намалението. Не е задължително, не слагайте на всички продукти. В същия формат като новата цена
    "is_featured": 1,                          // специален продукт, ще се показват на началната страница. Слагайте го рядко, на 4-5 продукта, иначе не го пишете
    "category_id": 1                           // категорията. id на категорията съответства с id на категорията от categories.json
}
// текста се огражда в кавички. Числата може и без тях.

в създадената папка imeImages добавяте и изображенията, които са свързани с артикула и се показват в слайдера на продукта.
Създавате нов файл ime_images.json и там пишете изображенията на продукта.
{
    "url": "/public/uploads/url",            // изображението. Формата е следния: "/public/uploads/ime_na_izobrajenieto.razshirenie"
    "product_id": 1                          // това е id на артикула. Тука пишете id на продукта който сте добавили.
}
ако сред изображенията я има и основната снимка на продукта, не я качвате повторно в папката
